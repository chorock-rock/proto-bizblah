## 오늘 (2026년 2월 2일) 변경사항

### 1. 브랜드 리뷰 기능 추가
- 점주들이 자신의 브랜드를 평가할 수 있는 리뷰 기능 추가
- 브랜드별 평균 점수 및 세부 점수 확인 가능
- 5가지 평가 항목으로 브랜드 평가:
  - 수익성 만족도
  - 본사 지원 만족도
  - 물류·원가 합리성
  - 브랜드 경쟁력
  - 소통·정책 신뢰도
- 별점 입력 기능 (0.5점 단위, 최대 5점)
- 리뷰 작성 시 선택적 코멘트 작성 가능
- 다른 점주들의 리뷰 조회 가능
- 리뷰 작성 후에만 브랜드 리뷰 페이지 접근 가능

### 2. 검색 엔진 최적화 (SEO)
- 검색 엔진에서 사이트가 더 잘 검색되도록 최적화
- 소셜 미디어 공유 시 미리보기 이미지 및 설명 표시
- 각 페이지별로 적절한 제목과 설명 자동 설정
- 검색 엔진이 관리자 페이지는 크롤링하지 않도록 설정

### 3. 브랜드 리뷰 모바일 화면 개선
- 모바일 화면에서 평균 점수와 세부 점수가 **한 줄에 가로로 나란히 표시**됨
- 화면이 좁을 경우 가로 스크롤로 전체 내용 확인 가능
- 한 화면에서 평균 점수와 세부 점수를 동시에 확인 가능

### 4. 프로필 및 브랜드 선택 플로우 개선
- 프로필이 없고 브랜드도 없을 때 브랜드 선택 화면이 먼저 표시됨
- 브랜드를 선택한 후 닉네임 설정으로 이동
- 프로필이 없을 때 무조건 닉네임 설정으로 이동하던 문제 해결

### 5. 건의하기 기능 개선
- 건의하기 완료 후 "다시 건의하기" 버튼 표시
- 사용자가 자신의 건의사항 처리현황 및 접수내역 확인 가능
- "건의하기" / "내 건의내역" 탭으로 구분
- 각 건의사항의 상태 표시 (검토 중, 검토 완료, 처리 완료)
- 건의사항 상세 정보 확인 기능
- 실시간 상태 업데이트
- "검토 중" 상태 색상을 #ED4E39로 변경

### 6. 전체 색상 테마 변경
- 메인 컬러: #ED4E39 (빨간색 계열)
- 모든 버튼, 링크, 강조 요소에 일관된 색상 적용
- 그라데이션: #ED4E39 → #D63E2A
- 호버 효과 및 그림자도 메인 컬러 기반으로 통일
- 전역 CSS 변수 업데이트 (`src/styles/colors.css`)

### 7. 로고 변경 및 크기 조정
- SVG 형식의 비즈블라 로고 사용
- 모든 화면에서 일관된 로고 표시
- 로고 크기 축소 (데스크톱: 36px, 모바일: 28px)
- 벡터 형식으로 확대 시에도 선명함 유지

### 8. 브랜드 리뷰 접근성 개선
- 리뷰를 작성하지 않은 사용자에게 "리뷰 작성하기" 버튼 표시
- "리뷰 작성 후 이용 가능합니다" 메시지와 함께 바로 작성 가능

### 9. 네비게이션 버튼 디자인 개선
- "← 목록으로" 버튼에 배경색과 테두리 추가
- 호버 시 색상 변경 효과
- 더 명확하고 클릭하기 쉬운 디자인
- 호버 시 이동(translate) 애니메이션 제거

### 10. 닉네임 자동 생성 기능
- 닉네임 설정 화면 진입 시 자동으로 고유한 닉네임 생성
- 영어 단어 + 숫자 조합 (예: "zap1234", "blip5678")
- Firestore에서 중복 확인하여 고유성 보장
- 주사위 버튼(🎲)으로 새 닉네임 재생성 가능
- 생성 중 로딩 텍스트 표시 없이 항상 🎲 아이콘 유지
- 사용자가 직접 입력하거나 자동 생성된 닉네임 사용 가능
- 다양한 랜덤 영어 단어 사용 (zap, blip, flip, snap 등)

### 11. 웰컴 화면 추가
- 첫 방문 시 웰컴 화면 표시
- 비즈블라 로고 및 서비스 소개
- "혼자 고민하지 마세요" 메시지
- "점주끼리 실시간으로" 강조 표시 (하단 밑줄 추가)
- "지금 시작하기" 버튼으로 브랜드 선택 화면으로 이동
- highlight 클래스에 밑줄 스타일 추가 (`border-bottom: 2px solid #ED4E39`)

### 12. 브랜드 선택 화면 개선
- 검색 기능이 있는 커스텀 드롭다운 구현
- 브랜드명 검색으로 빠른 선택 가능
- "내 브랜드가 없어요" 옵션을 셀렉트 박스에서 제거하고 하단에 별도 버튼으로 배치
- 버튼 텍스트: "내 브랜드 추가하기"
- 커스텀 입력 모드에서 셀렉트 박스 클릭 시 일반 모드로 전환
- 새로운 브랜드 리스트 (10개 브랜드):
  - 메가엠지씨커피
  - 컴포즈커피
  - 이디야커피
  - 스타벅스
  - 빽다방
  - 투썸플레이스
  - 더벤티
  - 텐퍼센트스페셜티커피
  - 매머드커피
  - 매머드익스프레스
- 플레이스홀더 텍스트: "브랜드를 검색하거나 선택해주세요"

### 13. Login 페이지 디자인 개선
- 심플하고 깔끔한 디자인
- "{브랜드명} 점주님, 안녕하세요." 큰 제목으로 표시
- "Google 계정으로 간편하게 로그인하고 익명 커뮤니티를 시작해보세요." 설명 추가
- 로고 및 불필요한 요소 제거
- 흰색 배경에 중앙 정렬 레이아웃
- 카드 스타일 제거

### 14. 텍스트 변경
- "프랜차이즈" → "자영업자" (전체 앱에서 일괄 변경)
- "내 브랜드가 없어요." → "내 브랜드 추가하기"
- "브랜드를 선택하세요" → "브랜드를 검색하거나 선택해주세요"
- SEO 메타 태그 및 설명에서도 "자영업자"로 통일

### 15. 홈 화면 게시글 표시 개선
- 홈 화면("all" 뷰)에서 브랜드 필터링 제거
- 모든 브랜드의 게시글을 전체 표시
- "내가 쓴 글" 뷰는 여전히 사용자별로 필터링
- Board 컴포넌트의 쿼리 로직 수정

### 16. 관리자 대시보드 차트 색상 차별화
- "가입자" 라인: #ED4E39 (빨간색)
- "게시글" 라인: #4285F4 (파란색)
- 두 라인의 색상을 명확히 구분하여 가독성 향상

### 17. 헤더 로고 클릭 시 스크롤 기능
- 홈 화면에서 스크롤을 내렸을 때 헤더의 "비즈블라" 로고 클릭 시 맨 상단으로 부드럽게 스크롤
- 게시글 상세 페이지에서는 홈으로 이동 후 스크롤
- `window.scrollTo({ top: 0, behavior: 'smooth' })` 사용

### 18. 배경색 변경
- "점주님 안녕하세요!" 페이지(Login 페이지) 배경을 흰색으로 변경
- 메인 콘텐츠 영역 배경을 흰색으로 변경
- 전역 body 배경색을 흰색으로 설정
