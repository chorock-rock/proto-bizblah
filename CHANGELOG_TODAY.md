# 오늘 추가된 변경사항 (To-be → As-is)

## 1. 브랜드 리뷰 기능 추가

### To-be (변경 후)
- 점주들이 자신의 브랜드를 평가할 수 있는 리뷰 기능 추가
- 브랜드별 평균 점수 및 세부 점수 확인 가능
- 5가지 평가 항목으로 브랜드 평가:
  - 수익성 만족도
  - 본사 지원 만족도
  - 물류·원가 합리성
  - 브랜드 경쟁력
  - 소통·정책 신뢰도
- 별점 입력 기능 (0.5점 단위, 최대 5점)
- 리뷰 작성 시 선택적 코멘트 작성 가능
- 다른 점주들의 리뷰 조회 가능
- 리뷰 작성 후에만 브랜드 리뷰 페이지 접근 가능

### As-is (변경 전)
- 브랜드 리뷰 기능 없음
- 점주들이 브랜드를 평가할 수 있는 기능 부재

## 2. 검색 엔진 최적화 (SEO)

### To-be (변경 후)
- 검색 엔진에서 사이트가 더 잘 검색되도록 최적화
- 소셜 미디어 공유 시 미리보기 이미지 및 설명 표시
- 각 페이지별로 적절한 제목과 설명 자동 설정
- 검색 엔진이 관리자 페이지는 크롤링하지 않도록 설정

### As-is (변경 전)
- 검색 엔진 최적화 없음
- 소셜 미디어 공유 시 미리보기 정보 없음

## 3. 브랜드 리뷰 모바일 화면 개선

## 4. 프로필 및 브랜드 선택 플로우 개선

### To-be (변경 후)
- 프로필이 없고 브랜드도 없을 때 브랜드 선택 화면이 먼저 표시됨

### As-is (변경 전)

## 5. 건의하기 기능 개선

### To-be (변경 후)
- 건의하기 완료 후 "다시 건의하기" 버튼 표시
- 사용자가 자신의 건의사항 처리현황 및 접수내역 확인 가능
- "건의하기" / "내 건의내역" 탭으로 구분
- 각 건의사항의 상태 표시 (검토 중, 검토 완료, 처리 완료)
- 건의사항 상세 정보 확인 기능
- 실시간 상태 업데이트

### As-is (변경 전)
- 건의하기 완료 후 다시 작성하기 어려움
- 자신이 제출한 건의사항 확인 불가
- 처리현황 확인 불가

## 10. 닉네임 자동 생성 기능

### To-be (변경 후)
- 닉네임 설정 화면 진입 시 자동으로 고유한 닉네임 생성
- 형용사 + 명사 + 숫자 조합 (예: "용감한사자1234")
- Firestore에서 중복 확인하여 고유성 보장
- 주사위 버튼으로 새 닉네임 재생성 가능
- 사용자가 직접 입력하거나 자동 생성된 닉네임 사용 가능

### As-is (변경 전)
- 사용자가 직접 닉네임을 입력해야 함
- 중복 확인 후 수동으로 다른 닉네임 입력 필요

## 12. 브랜드 선택 화면 개선

### To-be (변경 후)
- 검색 기능이 있는 커스텀 드롭다운 구현
- 브랜드명 검색으로 빠른 선택 가능
- "내 브랜드가 없어요" 옵션을 셀렉트 박스에서 제거하고 하단에 별도 버튼으로 배치
- 커스텀 입력 모드에서 셀렉트 박스 클릭 시 일반 모드로 전환
- 새로운 브랜드 리스트 (10개 브랜드):
  - 메가엠지씨커피, 컴포즈커피, 이디야커피, 스타벅스, 빽다방, 투썸플레이스, 더벤티, 텐퍼센트스페셜티커피, 매머드커피, 매머드익스프레스

### As-is (변경 전)
- 기본 HTML select 박스 사용
- 검색 기능 없음
- "내 브랜드가 없어요"가 셀렉트 박스 옵션으로 포함됨
- 기존 브랜드 리스트 (10개 브랜드)
